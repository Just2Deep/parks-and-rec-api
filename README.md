# API Details ğŸ 

Base URL - [https://parks-and-rec-api.frozencoast05.workers.dev](https://parks-and-rec-api.frozencoast05.workers.dev)

## Get random Quote

Request Path - [/quote/random](https://parks-and-rec-api.frozencoast05.workers.dev/quote/random)

Response -

```json
{
    "id": 27,
    "character": "Ben",
    "quote": "Bababooey."
}
```
